# APSP_Rust
A Rust program to solve the all pairs shortest path problem
It contains six algorithm:
- Floyd-Warshall (pethraph)
- Floyd-Warshall (own implementation)
- Dijsktra
- Dijkstra parallel
- Floyd-Warshall Blocked
- Floyd-Warshall Parallel
